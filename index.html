01 Overview

This script performs a complete analysis of vehicle fuel efficiency using the Auto.csv dataset. It covers exploratory data analysis, correlation study, time-trend modeling, group comparisons via ANOVA, linear regression, and two classification models (KNN and Random Forest) predicting whether a car's fuel efficiency is High or Low relative to the dataset median.

03 Installation

Install all required packages with a single command in your R console:

install.packages(c("tidyverse", "caret", "randomForest", "class", "corrplot"))R
Note: The install line is commented out in the script. Run it once in your R console before sourcing the script for the first time.

04 Usage

Place Auto.csv in your R working directory, then source the script:

# Set working directory to where Auto.csv lives
setwd("/path/to/your/data")

# Run the full pipeline
source("auto.R")R

The script reads Auto.csv directly — no file picker is used. All plots render to the active R graphics device and all model summaries print to the console.

Input file: Auto.csv must be in the working directory. The dataset should contain columns for mpg, cylinders, displacement, horsepower, weight, acceleration, year, and origin.

05 Pipeline

  • 1
    Load Data Reads Auto.csv with stringsAsFactors = FALSE. Prints structure and summary statistics.
  • 2
    Data Cleaning Replaces "?" values in horsepower with NA, converts it to numeric, removes all rows with missing values, casts origin as a factor, and derives a binary mpg_category (High / Low) split at the median.
  • 3
    Correlation Matrix Computes Pearson correlations across all 7 numeric variables and renders a circle corrplot.
  • 4
    Scatter Plots Two base-R scatter plots: Weight vs MPG and Horsepower vs MPG.
  • 5
    Time Trend Analysis Plots MPG over model year with a fitted OLS trend line in red to visualise efficiency improvements over time.
  • 6
    Boxplots Group comparisons of MPG by number of cylinders and by region of origin.
  • 7
    ANOVA One-way ANOVA testing whether mean MPG differs significantly across cylinder groups.
  • 8
    Linear Regression OLS regression of mpg on all 7 predictors. Outputs coefficients, R², and significance tests.
  • 9
    Train / Test Split Stratified 70/30 split on mpg_category using set.seed(123) for reproducibility.
  • 10
    KNN Classification Features are z-score scaled. K-Nearest Neighbours with k = 5 predicts High/Low MPG on the test set.
  • 11
    Random Forest Classification Trains a Random Forest with 500 trees (ntree = 500) on all 7 predictors including the origin factor. Evaluates on the test set and plots variable importance.

06 Variables & Features

Features used across models:

cylindersnumeric
displacementnumeric
horsepowernumeric*
weightnumeric
accelerationnumeric
yearnumeric
originfactor
mpgtarget (regression)
mpg_categorytarget (classification)

* horsepower contains "?" entries in the raw CSV which are coerced to NA and removed during cleaning.

Target Variable

mpg_category is derived at runtime by splitting mpg at the dataset median. Cars above the median are labelled High; those at or below are labelled Low.

07 Models

Linear Regression

lm(mpg ~ cylinders + displacement + horsepower +
        weight + acceleration + year + origin,
   data = auto)R

Predicts continuous mpg. Outputs coefficient table, adjusted R², and F-statistic.

K-Nearest Neighbours (KNN)

knn(train_x, test_x, train_y, k = 5)R

Classifies mpg_category. All 6 numeric predictors are z-score scaled before fitting. No origin factor (numeric features only).

Random Forest

randomForest(mpg_category ~ cylinders + displacement +
               horsepower + weight + acceleration +
               year + origin,
             data = train, ntree = 500)R

Classifies mpg_category using all 7 predictors including origin. Produces a variable importance plot via varImpPlot().
